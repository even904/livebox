# livebox
Multiple media box with pixel screen.

## 一、项目简介
本项目为一个多功能像素显示器。

## 二、开发软硬件环境
1. 主控使用STM32F103C8T6单片机
2. 像素屏幕为自制，使用SK6812灯带排列为16*16点阵
3. 主要使用CUBEMX+Keil5开发
4. 部分图像转换脚本为python（未完善，暂未上传）

## 三、已经实现的功能
#### 底层

#### 模块
1. SK6812与ws2812驱动方式基本相同，使用PWM+DMA驱动点亮
2. 实现任意画点、画圆（限于像素数量效果一般）
3. RGB图片显示（需用脚本转换）
4. 简单动画显示
5. 数字显示
